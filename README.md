# 🎬 CinemaVerse

CinemaVerse is a modern full-stack cinema ticket booking platform built with ASP.NET Core Web API (.NET 9) following clean architecture and scalable backend engineering practices.

> ⚠️ Backend development is completed.  
> 🚧 Frontend application is currently under active development.

---

## ✨ Features

- User Authentication & Authorization using JWT
- Role-Based Access Control (Admin / User)
- Secure Password Hashing with BCrypt
- Movie & Showtime Management
- Cinema Seat Reservation System
- Stripe Payment Integration
- Background Jobs using Hangfire
- Email Notifications System
- Global Exception Handling
- Swagger API Documentation
- In-Memory Caching
- Logging with Serilog

---

## 🛠️ Tech Stack

### Backend
- ASP.NET Core Web API (.NET 9)
- Entity Framework Core 9
- SQL Server
- JWT Authentication
- Hangfire
- Stripe
- Serilog
- MailKit & RazorLight

### Architecture & Practices
- N-Tier Architecture
- Repository Pattern

- Dependency Injection
- Async/Await Programming
- Rate Limiting
- Clean Code Principles
